# Catalogue HSK 3.0

`hsk30-source.csv` contient les 11 092 entrées du référentiel HSK 3.0 (hanzi simplifié et traditionnel, pinyin, catégorie, niveau et références CEDICT). Source : [ivankra/hsk30](https://github.com/ivankra/hsk30), données dérivées du standard chinois 2021 et dépôt sous licence MIT.

Les colonnes de traduction française, d’exemples et de thèmes pédagogiques sont prévues dans la table D1 `vocabulary_words`. Les fiches enrichies visibles dans la première version couvrent les niveaux 1–3 ; le fichier source garantit la disponibilité hors-ligne du catalogue complet pour les importations D1 suivantes.
